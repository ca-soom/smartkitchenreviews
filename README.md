# SmartKitchenReviews — SEO Money Site

Air Fryer niche static site built for monetization via Amazon Associates.

## 🚀 Quick Deploy to GitHub Pages

```bash
cd deploy
git init && git add . && git commit -m "Initial deploy"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/smartkitchenreviews.git
git push -u origin main
# Then: Settings → Pages → Source: main → Save
```

## 📁 Structure
- `index.html` — Homepage with all article cards
- 8 article pages covering Best Of, VS, How To, Problem Solver
- `sitemap.xml` + `robots.txt` for SEO
- `404.html` — Custom error page

## 💰 Monetization
- Amazon Associates affiliate links (tag: `smartkitchen-20` — replace with yours)
- Google Analytics + AdSense placeholders ready
